## Install

Clone atau download repository
```php
https://github.com/naufal1122/Project_FoodCourt_23
```

Install database .sql di dalam local
```php
foodcourt2(1).sql
```

Edit konfigurasi di .env
```php
.env
```

<a name="section-2"></a>
## Menjalankan

Jalankan di terminal
```php
php artisan serve
```

Jalankan di browser
```php
http://localhost:8000
```
